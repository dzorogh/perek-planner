Шапка и навигация — варианты

Soft Workshop / Lavender уже зафиксированы. Здесь только структура chrome: где живут шаги плана, что такое «Дни», и как выглядит бренд в шапке. Выбери буквы (например: W2 · D1 · L3).

1 · Куда уезжает wizard (шаги плана)
W1

Stepper под глобальной шапкой

Глобальный chrome всегда один. Шаги появляются только на экранах активного плана — второй ряд, сразу под шапкой.

p keplo
Создать менюИсторияНастройкиВыйти
Меню Порции Список
Контент плана… История / Настройки — без stepper.

Плюс: привычный паттерн wizard. Минус: +высота chrome на каждом шаге.

W2

Stepper в теле страницы (page chrome)

Шапка только глобальная. Шаги — заголовок + stepper внутри контента плана, как у страницы «Порции» / «Список».

Keplo
Создать менюИсторияНастройкиВыйти
План порций Меню Порции Список
Контент…

Плюс: шапка чистая всегда. Минус: stepper «уезжает» при скролле (если не sticky).

W3

Только Continue / Back — без постоянного stepper

Нет ряда «Меню · Порции · Список». Линейный поток: кнопка «Дальше» внизу, «Назад» рядом. Прогресс — текстом («Шаг 2 из 3»).

Keplo
Создать менюИсторияНастройкиВыйти
Шаг 2 из 3 · Порции
Контент… внизу: ← Меню · К списку

Плюс: минимум chrome. Минус: хуже прыгать между шагами; UJ-1 gate всё равно нужен.

W4

Контекстный «текущий план» в шапке

В шапке — бренд + глобальные ссылки. Если есть активный план — компактная ссылка «Текущий план · Порции ›». Полный stepper — внутри плана (как W1 или W2).

Keplo
План · Порции › Создать меню История Выйти
На Истории: нет подсветки «Дни». Возврат в план — одной ссылкой.

Плюс: решает «где я?» вне визарда. Минус: чуть богаче глобальная шапка (ты сказал «вроде хватает» — это опциональное расширение).

2 · Что такое «Дни» / Create Menu
D1

Создание — вне визарда

«Создать меню» = отдельный экран (длина + люди + Сгенерировать). Wizard начинается после генерации: Меню → Порции → Список. «Дни» в stepper нет.

Keplo
Создать менюИстория
Новое меню
1–4 дня · люди · Сгенерировать
↓ после успеха → Меню

Ближе к твоей формулировке «кнопка Создать → шаги в другом месте».

D2

«Дни» — шаг 0 визарда

Тот же экран создания, но он первый в stepper. После генерации активны следующие шаги. На Истории stepper скрыт.

Дни Меню Порции Список
Создание = шаг визарда. Риск: снова смешать «создать» и «редактировать план».

Почти как сейчас по смыслу, но stepper не в глобальной шапке рядом с Историей.

D3

Landing = текущий план; Create — secondary

После логина — последний/активный план (или пустое состояние). «Создать меню» — primary CTA. «Дни» только внутри create-flow, не в основном stepper.

Keplo
Создать менюИстория
Пусто: «Нет активного меню» + CTA
Есть план: сразу сетка Меню / stepper

Меняет post-login landing (сейчас Create). Сильнее сдвиг IA.

3 · Бренд в шапке (сейчас — строка)
L1

Только wordmark (уточнённый)

Без иконки. Типографика сильнее: вес / трекинг / одна строка. Подзаголовок опционален или убирается.

Keplo
опционально без «Планировщик меню»
L2

Монограмма + wordmark

Простой знак «p» / «пп» в скруглённом квадрате primary + имя. Минимальный brand mark без иллюстрации.

p
Keplo
Планировщик меню
L3

Геометрический знак (порции / дни)

Абстрактная метка: 3 «слота» / сегмента — намёк на день×приём пищи. Не еда-эмодзи, не корзина магазина.

Keplo
Планировщик меню
L4

Знак-кольцо / штамп

Круглая монограмма — «штамп мастерской». Wordmark рядом. Чуть более «Soft Workshop», чем квадрат.

пп
Keplo
без подзаголовка — чище

Как ответить

Одна строка, например: W2 · D1 · L3. Можно комбинировать (W1+W4). Для design handoff без Stitch — напиши продюсера: Figma · v0 · другой (какой).
